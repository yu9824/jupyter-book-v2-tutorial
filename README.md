## Root Page

The â€œrootâ€ of a site is the page displayed when someone browses to the index of your site without any pathname. The CLI will choose the root file in the following order:

1. index.md / README.md / main.md
2. index.tex / README.tex / main.tex
3. index.ipynb / README.ipynb / main.ipynb
